
# Netflix Movies Data Analysis using Python

This project analyzes Netflix movies dataset using Python and Natural Language Processing (NLP) techniques.

## Project Description
The dataset was preprocessed and cleaned using Python libraries. Text data was converted to lowercase, tokenized, and cleaned by removing punctuation and stopwords. Stemming and lemmatization were applied to normalize words.

Advanced NLP techniques such as POS tagging and Named Entity Recognition (NER) were performed to extract meaningful information from movie descriptions. The cleaned data was prepared for further analysis and insights.

## Tech Stack
- Python
- Pandas
- NLTK
- NLP (Text Preprocessing)
